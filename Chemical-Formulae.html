Chemical Equations

Here are some important Chemical Equations:

  1. H2 + O2 → H2O
  2. C + O2 → CO2
  3. N2 + 3H2 → 2NH3
  4. CH4 + 2O2 → CO2 + 2H2O
  5. Ca + 2H2O → Ca(OH)2 + H2
  6. 2Na + Cl2 → 2NaCl
  7. S + O2 → SO2
  8. Fe + S → FeS
  9. C6H12O6 → 2C2H5OH + 2CO2
  10. Zn + 2HCl → ZnCl2 + H2
