Mathematical Formulae

Here are some important mathematical Formulae:

  1. (a+b)2 = a2 + 2ab + b2
  2. (a-b)2 = a2 - 2ab + b2
  3. a2 - b2 = (a-b)(a+b)
  4. a2 + b2 = (a+b)2 - 2ab
  5. a3 + b3 = (a+b)(a2 - ab + b2)
  6. a3 - b3 = (a-b)(a2 + ab + b2)
  7. (a+b)3 = a3 + 3a2b + 3ab2 + b3
  8. (a-b)3 = a3 - 3a2b + 3ab2 - b3
  9. (a+b+c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca
  10. (x+a)(x+b) = x2 + (a+b)x + ab
